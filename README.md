<h2>Hướng dẫn cách Setup Project</h2>
<p>Sau khi clone project bạn cần chạy lệnh dưới đây để hoàn tất quá trình setup</p>

Đối với hệ điều hành Linux:

```
./setup.sh
```

Đối với hệ điều hành Windows:

```
start setup.exe
```
